Citation file explanations:

- citation_matrix.csv: 
    Contains the galaxy_name and it's associated unique citation IDs from 'citations.bib'.

- citations.bib: 
    The Bibtex file with all the citations and their unique citation ID.

- compiled_citations_document.pdf: 
    Compiled version of 'citations.bib'. Use this as a reference for the citation style.

- Mannheim, O'Brien 2012.pdf:
    This is the article that all of the SOCM data and citations came from. Use this as a reference for how the SOCM 'velocities_citation', 'luminosity_citation', 'scale_length_citation', and 'mass_hydrogen_citation' columns should look like.
